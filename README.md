# Informatik Alchemie – Klasse 7

Ein kleines Browser-Spiel für den Informatikunterricht in Klasse 7.

Die Schülerinnen und Schüler kombinieren zwei Begriffe und entdecken daraus neue Informatik-Konzepte. Das Spiel ist bewusst wie ein kleines „Little Alchemy“ aufgebaut und eignet sich zum Wiederholen, Vernetzen und Besprechen von Grundbegriffen.

## Start

Die App besteht aus statischen Dateien und läuft direkt im Browser:

- `index.html` öffnen
- oder über GitHub Pages veröffentlichen

## Dateien

- `index.html` – Grundstruktur der App
- `style.css` – Gestaltung, Layout und Hintergrund
- `script.js` – Spiellogik, Kombinationen und Speicherung

## Funktionen

- 57 Informatik-Konzepte
- Startbildschirm mit kurzer Erklärung
- Fortschrittsanzeige
- Levelanzeige
- Kategorienfilter
- Suchfeld
- Tipp-Funktion
- automatischer Hinweis nach mehreren Fehlversuchen
- Lexikon mit kurzen Erklärungen
- fachliche Hinweise bei vereinfachten Kombinationen
- optionaler Ton
- Speicherung des Fortschritts mit `localStorage`

## Pädagogischer Hinweis

Einige Kombinationen sind spielerisch vereinfacht. Beispiel: Ein Byte besteht fachlich aus 8 Bit. Im Spiel steht `Bit + Bit = Byte` als didaktische Vereinfachung für den Übergang von einzelnen Bits zu größeren Dateneinheiten.

Das Spiel eignet sich deshalb besonders als Einstieg, Wiederholung oder Gesprächsanlass, nicht als vollständige Fachsystematik.

## Einsatz im Unterricht

Mögliche Nutzung:

1. Schülerinnen und Schüler spielen allein oder zu zweit.
2. Gefundene Begriffe werden im Heft gesammelt.
3. Schwierige Kombinationen werden gemeinsam besprochen.
4. Das Lexikon dient als kurze Wiederholung.

## Technischer Stand

Die App nutzt nur HTML, CSS und JavaScript. Es sind keine externen Bibliotheken und kein Login nötig.
